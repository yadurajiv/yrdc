Standalone visual system

A quiet, paper-first interface language.

Vellum System is a portable CSS package for warm, focused interfaces. It is organized into tokens, atoms, molecules, organisms, templates, and pages so humans and agents can compose consistent screens from small, understandable parts.

Use one entrypoint: vellum-system/vellum.css. Add class="vellum" to the body or a scoped wrapper, then compose from the layers below.
