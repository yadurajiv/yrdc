Trivial Engine

Nov 7th 2011 By Yadu Rajiv Updated Jun 5th 2026
Trivial Engine is a game framework inspired by game frameworks like Flixel and FlashPunk. The goal for Trivial is to make writing games easier and simple, giving you a higher level framework and tools to work with so that you can focus on what you do best, make games. Trivial Engine is now free and open source. Getting our base app up and running is easy as, with 3 lines of code.
Trivial::App *myApp = Trivial::App::Instance();
myApp->configure(NULL, 800, 600, 32, "my game");
myApp->run();
Trivial Engine repository: https://bitbucket.org/hashstash/trivial-engine
