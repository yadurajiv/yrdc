surprise!!

Nov 3rd 2006 By Yadu Rajiv Updated Jun 5th 2026
Today was a long day, and it really has become tomorrow.. So.. to be a bit more accurate.. yesterday was a long day.. and it was a nice day..

Work has finally resumed after almost a month of hanging around..

Apart from work.. something new that i tried was the ftp option in blogger!! which was implemented only now ( and blogger beta is finally feature complete ).. now I'm hosting my own blogs.. :) that's happy news :P sort of..

The second thing even hit me as a surprise.. I downloaded and installed Internet Explorer 7.. the 14+ MB download was almost effortless.. the installation was terribly long!! but i cant help but say that it was worth the wait.. This without any doubt is the best version of IE out there to date!! The setup and updates were long but totally unmanned and effortless (if u don't count the wait )..

Even if Opera came out with tabs.. firefox with extensions.. and other nice additions.. IE7 is a really nice experience.. It gives you maximum viewing area.. tabbed browsing.. nice search options.. a quick tab view, which is a thumbnail view of all tabs.. and an absence of a file menu!! which i really liked.. some elements of the menu has become a small toolbar menu.. With the cleartype options on, ie renders text very very very well!! The memory it uses might be a bit on the heavy side, and increasing with an increase in tabs.. But this is the case with ff2 as well..

Since I am really used to the idea of a spell check in the browser, which was one of the latest features of firefox2, i would really like to see one here too.. If i wont use ie as my primary browser, it would be because of the lost faith in the browser, getting that back would be a tough job.. but they sure as hell is on the right track!!

Thats it for today.. good night and good luck :)

ps: all of a sudden i see a small button that says upload file!! thats new too!! mustve come up because im hosting the blog i guess.. hmmm...
